# Audio-Generation-using-Neutral-Net

## Audio Generation Using :
* LSTM
* LSTM using Embedding
* GAN

## LSTM parameters used :
- Epoch : 70
- Dropout : 0.5
- Activation function : Relu (dense: 3)
- Loss : Mae
- Optimizer : Adam
- Batch size : 256

## Embedded LSTM parameters :
- 128 notes
- Embed size : 100
- Dropout : 0.3
- Activation function : Relu (dense: 1)
- Optimizer : RMSProp

## GAN parameters : 
- 128 notes
